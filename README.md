# Welcome to Pinnacle-ide.


Pinnacle-ide is the open-source Cursor alternative.

Use AI agents on your codebase, checkpoint and visualize changes, and bring any model or host locally. Pinnacle-ide sends messages directly to providers without retaining your data.

This repo contains the full sourcecode for Pinnacle-ide. If you're new, welcome!
